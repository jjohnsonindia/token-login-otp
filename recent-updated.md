
### 🔐 Security & Access Control Enhancements (19 March 2026)

#### 1. Sensitive Data Access Restriction
- Restricted access to the following student details based on user roles:
  - Community
  - Caste
  - Religion
  - Student Mobile Number
  - Student Email ID
- Only users with proper permissions can view these details.

#### 2. Editing Restrictions Implemented
- Editing access is now restricted (based on user role) for:
  - Student Name
  - Register Number
  - Admission Number
  - Admission Date
- Prevents unauthorized modifications to critical student data.

---

### ⚙️ Configuration

- All above settings are available under:
  **Student_Basic → Side Modules Menu**
- Only authorized users can access these options.
- Institution Admin can configure and manage role permissions.

---

### 📝 Notes
- These updates improve data security and ensure controlled access across the ERP.
